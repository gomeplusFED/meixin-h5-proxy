[meixin-h5-proxy](https://github.com/yanglang1987500/meixin-h5-proxy) ― 美信H5 CDN代理模块
==================================================

简介
----

此模块通过nodejs安装后可以直接进行CDN资源代理，便于调试。

使用指南
----
与gulp一样，支持全局命令proxy<br>

```
npm install meixin-h5-proxy -g
```
```
npm install meixin-h5-proxy --save-dev
```
